# 汽水音乐自动获取免费额度

这是一个Python项目，通过adb命令操作android手机，完成自动获取汽水音乐看广告的功能：

1、检查系统是否支持adb，不支持提示退出； 

2、adb devices 检查是否有设备连接，无连接设备则提示退出； 

3、adb shell getprop ro.product.model 命令获取手机型号

4、通过 Adb shell 命令获取当前启用的应用，需要为“汽水音乐”——com.luna.music/com.ss.android.excitingvideo.ExcitingVideoActivity，否则循环提示，等待用户进入“汽水音乐->免费”的界面；

5、检查运行目录下是否有手机型号对应的图像目录；没有则使用"default"目录（使用defaut目录需要提示）（这个目录保存了从对应手机截屏中截取的部分特征部分），加载目录下的图片到 内存

6、adb 命令截屏，保存图像；

7、通过图像模板匹配的方法，分别检查截屏图像中存在是否存在手机型号对应中的图片，如果置信度高于某个阈值，默认0.8，通过adb命令点击图像所在位置的中间位置；

8、每隔2秒，循环4、5任务，如果持续30次没有点击动作发生，退出程序。
